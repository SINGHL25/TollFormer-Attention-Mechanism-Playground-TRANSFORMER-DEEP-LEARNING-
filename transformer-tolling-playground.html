Tolling Context — Why Attention?

A toll system processes sequences of events: vehicle entry → lane selection → payment → exit. Just like the word "mole" changes meaning with context, a truck token means different things near peak-hour vs off-peak. Attention lets the model relate any event to any other — even events far apart in time.

🚗 Token = Toll Event

Each vehicle passing a toll plaza becomes a token — its type, speed, lane, time all encoded into a single high-dimensional vector called an embedding.

🔗 Context Window

The model sees a window of recent events simultaneously — just like a human dispatcher watching the last hour of footage — and each event can attend to all others.

🎯 Prediction Goal

Given a sequence of toll events, predict: next vehicle class, congestion level, fraud probability, or optimal lane routing — all from the final embedding vector.

⚡ Parallelism

All attention heads run simultaneously on GPUs — this massive parallelism is WHY transformers scaled so far, enabling real-time toll analytics at highway scale.

STEP 2B

Query · Key · Value Matrices

The three learnable matrices that power attention. Each token generates Q, K, V vectors via matrix multiplication.

Attention(Q,K,V) = softmax(QKᵀ / √dₖ) · V

STEP 2C

Causal Masking

During training, future toll events cannot influence past ones. The upper triangle is masked to -∞ before softmax.

Attention Mask Visualizer
■ ALLOWED — can attend (past/present)  ■ MASKED — set to -∞ (future)
Click any cell to see why it's masked/allowed
Why Masking Matters
Before softmax: set upper triangle = -∞
After softmax: e^(-∞) = 0
🔴 Without masking: Training example at time T could "cheat" by reading T+1 data — the model memorizes answers instead of learning patterns.
🟢 With masking: Each prediction only uses past context. One sequence = N training examples simultaneously!
🚗 Tolling context: When classifying vehicle at position T, only use entries T-1, T-2... Can't use data from vehicles that haven't arrived yet!
Masking efficiency gain:
Without masking1 example
With masking (N=8 tokens)8 examples
